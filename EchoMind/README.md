# 🔵 EchoMind — AI That Remembers You

## What is this?
A conversational AI chatbot with short-term memory.

## Tech Stack
- Python 3.11
- Groq API (LLaMA 3)
- LangChain

## How to Run
1. Clone the repo
2. pip install -r requirements.txt
3. Add GROQ_API_KEY to .env
4. python app/main.py
